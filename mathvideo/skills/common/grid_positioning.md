# 网格定位系统使用技巧

## 核心原则
- 始终在 ORIGIN 构建几何图形，然后用 `place_in_area()` 定位
- 大的几何图形使用 `place_in_area(obj, 'B2', 'I8')`，预留顶部和底部空间给标签
- 小对象（标签、点）使用 `place_at_grid(obj, 'E5')`

## 常见错误
- **不要** 在 `place_in_area()` 后再对子对象调用 `place_at_grid()`，后者会覆盖前者的定位
- **不要** 使用 `.to_edge()` / `.to_corner()` / 绝对坐标

## 推荐区域划分
- 标题区: A1-A10
- 主体图形: B2-I8（留边距）
- 底部注释: J1-J10
